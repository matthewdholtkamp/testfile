        How we got here

        Timeline from mechanism map to decision slate

        This is the project path that led to the current decisions. Read it top to bottom when you want the short version of what each phase contributed.

        2026-03-29 02:26:55
        Phase 1 · Process Lanes supported

        The engine reduced the TBI literature into six recurring process lanes so the project stopped treating every paper as a one-off mechanism note.

        Why it matters now: Today’s decisions only make sense because the lane map already tells us which biology is actually in play. Latest repo milestone: 6 lanes
        2026-03-28
        Phase 2 · Causal Transitions supported

        The engine converted lane-level mechanisms into explicit causal bridges, so the system can say what likely drives what next.

        Why it matters now: Current hinge and leverage decisions are built on these bridges. Latest repo milestone: Complete starter-lane coverage in Phase 2 transitions
        2026-03-28
        Phase 3 · Progression Objects supported

        The engine grouped recurring downstream damage states into progression objects such as chronic network dysfunction and neurovascular uncoupling.

        Why it matters now: The current decision slate uses these objects to explain why a choice matters downstream. Latest repo milestone: Add Phase 3 progression object layer
        2026-03-28
        Phase 4 · Translational Logic bounded

        The engine attached translational packets to each lane so targets, readouts, and intervention logic sit on top of the mechanism map.

        Why it matters now: Biomarker panels and leverage-point choices come directly from these packets. Latest repo milestone: Implement Phase 4 translational perturbation layer
        2026-03-28
        Phase 5 · Cohort Stratification usable

        The engine split TBI into cohort and endotype patterns so mild, repetitive, blast, severe, acute, and chronic states stop collapsing into one disease bucket.

        Why it matters now: The current ambiguity around tau versus clearance is an endotype problem, not just a mechanism problem. Latest repo milestone: Add Phase 5 cohort stratification layer
        2026-03-28
        Phase 6 · Hypothesis Rankings live

        The engine ranked the strongest bridges, weakest hinges, biomarker panels, leverage points, and next tasks into a weekly decision slate.

        Why it matters now: The portal now uses this slate as the human decision queue. Latest repo milestone: Implement phase 6 hypothesis rankings
